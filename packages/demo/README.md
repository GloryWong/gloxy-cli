## Usage

```
npm -g i @gs/demo
```

```bash
gs-demo --help

Usage: gs-demo [options] [command] [demoSelector]

Description: Instant and quick demo

Note:
  demoSelector: input a `demo code` from the displayed demo list
    * open a demo in a new editor window by default
    * -r: open the demo in the last active editor window
    * -a: archive the demo

Options:
  -V, --version    output the version number
  -l, --list       list all demos
  --create <name>  create a demo
  --tag <tags...>  use tags
  -h, --help       display help for command

Commands:
  init [path]      Init a GS Demo
  archive          Archive existing GS Demo
```
